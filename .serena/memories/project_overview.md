TCG Trove is a FastAPI + SQLAlchemy + Jinja2 university trading-card marketplace located at C:\Users\dimit\Desktop\computer_science\pythonprojects\tcgtrove. It was built by adapting the HomeFinder reference app without modifying HomeFinder. Core concepts: Listing now represents a card seller listing; location stores game/franchise; property_type stores rarity; furnished stores card condition; size stores stock quantity; bedrooms stores release year; bathrooms stores rarity score. Roles are buyer, seller, supervisor, admin. Added wallet/cart/checkout/order history and supervisor reports.
